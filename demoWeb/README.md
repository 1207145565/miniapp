## 虎牙小程序示例代码 - H5端

> - 包括以下小程序类型
> - web端互动通道面板
> - pc开播端互动通道面板
> - pc客户端互动通道面板

### 目录说明

* demoWeb3：源码目录
* build-result：可以直接拖拽到小程序程序配置区域进行配置
* demoWeb3.zip：可以直接作为源码包上传

### 使用说明

1. 在[ext.huya.com](https://ext.huya.com)上创建一个小程序，并创建一个版本
2. 在本地新建一个文件夹，把`demoWeb3`的内容拷贝到新建的文件夹中
3. 执行`npm i`
4. 开发执行`npx hyext start`
5. 打包执行`npx hyext release`
